The Ownership Challenge

Role: Internal Revenue Manager (Property Guardian)
Goal: Achieve GOP (Gross Operating Profit) target of 42%.

Budget season is the most critical time of the year. Working directly with the GM and Owners, I was responsible for forecasting revenue across all streams (Rooms, F&B, Spa) for the upcoming fiscal year. The goal was to set aggressive yet achievable targets.

Forecasting Model

I moved the property away from simple "Last Year + 5%" budgeting to a robust driver-based model. We analyzed segment pickup trends to forecast occupancy and ADR separately. 

# Budget Variance Logic
def calculate_variance(budget, actual):
    variance = actual - budget
    percent_var = (variance / budget) * 100
    
    if percent_var < -5.0:
        return "CRITICAL_MISS"
    elif percent_var > 5.0:
        return "EXCEEDED_TARGET"
    else:
        return "ON_TRACK"
