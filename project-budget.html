The Goal

Role: Revenue Manager
Goal: Reach a profit target of 42%.

Budgeting is very important. I worked with the General Manager and Owners to predict how much money we would make from Rooms, Food, and Spa for the next year.

How I Planned It

Instead of just adding 5% to last year's numbers, I looked at the trends. I checked how many rooms were booked and at what price for different types of guests. 

# Checking if we hit the budget
def calculate_variance(budget, actual):
    variance = actual - budget
    percent_var = (variance / budget) * 100
    
    if percent_var < -5.0:
        return "MISSED_TARGET"
    elif percent_var > 5.0:
        return "DID_GREAT"
    else:
        return "ON_TRACK"
