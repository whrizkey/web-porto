# Portfolio Website

A premium, responsive personal portfolio website customized for [User Name].

## Features
- **Premium Design**: Dark mode aesthetic with smooth animations.
- **Responsive**: Mobile-friendly layout with custom navigation.
- **Sections**: Hero, About, Projects (Grid), Contact.

## Project Structure
- `index.html`: Main entry point.
- `css/style.css`: All styles and animations.
- `js/app.js`: Interactivity (scroll effects, mobile menu).

## Deployment

### Option 1: Vercel (Recommended)
1.  Install Vercel CLI: `npm i -g vercel`
2.  Run `vercel` in this directory.
3.  Follow the prompts.

### Option 2: Netlify
1.  Drag and drop this folder into the Netlify dashboard.
2.  Or use the CLI: `netlify deploy`.

## Customization
- Edit `index.html` to update text and links.
- Add images to the `assets/` folder.
