# Portfolio Website

Responsive personal portfolio customized website.

# Features
- **Premium Design**: Dark mode aesthetic with smooth animations.
- **Responsive**: Mobile-friendly layout with custom navigation.
- **Sections**: Hero, About, Projects (Grid), Contact.

# Project Structure
- `index.html`: Main entry point.
- `css/style.css`: All styles and animations.
- `js/app.js`: Interactivity (scroll effects, mobile menu).
