Tracking Competitor Prices

Building a tool to see what other hotels are charging.

Download Source

The Problem

Role: Analyst
Goal: Make sure our prices are competitive.

Prices in Bali change all the time. I needed to make sure we weren't just following others, but setting the right price. Checking manually took too long.

My Solution

I used **Python** to automatically check prices from other websites. This helped us see if we were too cheap or too expensive. 

# Checking Competitor Prices
def track_compset_parity(our_rate, compset_avg):
    # We want to be 15% more expensive than the average
    target_price = compset_avg * 1.15
    
    if our_rate < target_price:
        action = "INCREASE_PRICE" # We are too cheap
    elif our_rate > (target_price * 1.1):
        action = "CHECK_VOLUME" # We might be too expensive
        
    return action

What happened

We stayed above the average price, which is good. But in Month 4, we got too close to the most expensive hotel, and fewer people booked. We fixed this by offering a special deal.
