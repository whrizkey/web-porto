The Price of Luxury: Hedonic Pricing Model

Reverse-engineering hotel pricing using NLP and Regression Analysis to determine the monetary value of room attributes.

View Python Script

The Question: "What is a Sea View Worth?"

Role: Data Scientist / Revenue Strategist
Tools: Python (scikit-learn, pandas, regex)
Technique: Natural Language Processing (NLP) & Linear Regression

Hotels often price rooms based on "gut feeling" or simplistic category multipliers. I wanted to replace intuition with data: Can we mathematically isolate the exact monetary value of specific attributes like "Breakfast", "Suite", or "Sea View"?

Methodology: Mining Value from Text

The raw data contained unstructured text descriptions. I used Regex and NLP techniques to feature-engineer binary flags (e.g., `has_view`, `is_suite`) and extracted numerical data (`sqm`) from these descriptions.

These features were then fed into a **Linear Regression model** to predict the Total Price. The coefficients of this model serve as the "Price Tag" for each attribute. 

# Feature Engineering & Modeling
def analyze_hedonic_pricing(df):
    # 1. NLP Extraction
    df['has_view'] = df['full_text'].apply(lambda x: 1 if 'sea' in x or 'view' in x else 0)
    df['is_suite'] = df['full_text'].apply(lambda x: 1 if 'suite' in x else 0)
    
    # 2. Linear Regression (Hedonic Model)
    model = LinearRegression()
    model.fit(df[['has_view', 'is_suite', 'has_breakfast']], df['price'])
    
    # 3. Validating the "Price Tag" (Coefficient)
    return model.coef_

Key Findings

  • The "Suite" Premium: Upgrading a guest to a Suite provides the highest value add (~24M IDR), justifying aggressive upsell paths for this inventory.
  • Club Access: Access to the Executive Club is valued at ~19M IDR, suggesting it is a major driver of willingness-to-pay for high-end travelers.
  • View Value: A view commands a ~2.3M IDR premium per night. This data can be used to dynamically price "High Floor" or "Ocean Front" upgrades during check-in.

Strategic Application

This model transforms how we approach pricing and marketing:

  • Attribute-Based Selling (ABS): Instead of selling generic "Room Types", we can unbundle features and sell them as add-ons (e.g., "Add Club Access for 15M").
  • Value-Based Marketing: We verify that "Breakfast included" packages are priced correctly relative to the perceived value (1.1M IDR).
